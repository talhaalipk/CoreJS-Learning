Basic Array Methods

  1. pop() -> Update org array --> return last index value AND remove it from org array
  2. push() -> Update org array -->return the new lenght AND add new element at the last of array
  3. shift() -> Update org array -->return the first element AND remove it from org array
  4. unshift() -> Update org array -->return the new lenght AND the element at o index of array
  5. concat() -> Do not Update org array --> return new array with both aray data
  6. indexOf(value) -> Do not Update org array --> return index of value in array
  7. includes(value) -> Do not Update org array --> retun true or false according to presence of value in array
  8. reverse() -> Update org array --> return the reverse array AND also reverse the org array
  9. sort() -> Update org array --> return the sort array AND also sort the org array

    10. Note

    This Sort Function sort num and string both in alphabtic order

  10. slice(s,e)-> Do not Update org array --> return the splice of array from (s to e-1)
  11. splice() -> Update org array --> return the splice of array from (s to e) and allso delete this part from org array
