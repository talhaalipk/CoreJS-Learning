Function


Types

  • Function Declaration -- > We can call it before its declaration
    • 
      function addNum(a,b)
      {
          return a+b ;
      }
  • Function Expression --> We can not call it before its declaration
    • 
        // Function Expression + Defination
        let addingTwoValues = function (a,b)
        {
            return a+b ;
        }
  • IIFE stands for Immediately Invoked Function Expression
