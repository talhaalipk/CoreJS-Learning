String Methods and Properties

Before starting string methods and properties we need know that strings are indexed. So basically we can access any single character of a string by doing something like this.

Expression: 'Hello World'[1]

Result: 'e'

There is only one property that exists.

Property

  • length

Methods (Without Argument)

  • toUpperCase()
  • toLowerCase()
  • trim() --> Remove Spaces Form Start and end Both
  • trimStart() --> Remove Spaces Form Start only
  • trimEnd() --> Remove Spaces Form end only

Methods (With Argument)

  • includes() --> return boolean value
  • indexOf() --> return starting index value
  • replace() --> replace First value
  • replaceAll() --> Replace all Values
  • concat() --> join 2 string
  • padStart()
  • padEnd()
  • charAt()
  • charCodeAt()
  • split() --> return array
